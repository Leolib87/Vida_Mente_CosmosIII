Guía de Comandos Unix/Linux

Una referencia rápida y completa para los comandos más utilizados en la terminal de Unix y Linux, con descripciones en inglés, español y ejemplos prácticos.

Basado en la guía de FOSSwire.com

File Commands / Comandos de Archivos

Estos comandos son esenciales para navegar y manipular archivos y directorios en el sistema.

ComandoDescripción (EN/ES)Ejemplo de Uso
ls -al Formatted listing with hidden files.Lista formateada incluyendo archivos ocultos. ls -al /home/usuario
cd dir Change directory to dir.Cambia al directorio 'dir'. cd /var/www
pwd Show current directory.Muestra el directorio actual. pwd
mkdir dir Create a directory dir.Crea un directorio llamado 'dir'. mkdir nuevo_proyecto
rm file Delete file.Elimina un archivo. rm archivo_temporal.txt
rm -rf dir Force remove directory dir.Elimina forzosamente un directorio y su contenido. rm -rf viejo_proyecto
cp f1 f2 Copy file1 to file2.Copia el archivo f1 a f2. cp config.txt config.bak
mv f1 f2 Rename or move file1 to file2.Renombra o mueve el archivo f1 a f2. mv reporte.doc /documentos/
ln -s file link Create symbolic link to file.Crea un enlace simbólico al archivo. ln -s /var/log/nginx/access.log web.log
touch file Create or update file.Crea un archivo vacío o actualiza su fecha de modificación. touch nuevo_fichero.txt
cat file Output the contents of file.Muestra el contenido de un archivo. cat /etc/hosts
head file Output the first 10 lines of file.Muestra las primeras 10 líneas de un archivo. head /var/log/syslog
tail -f file Output contents as it grows.Muestra el contenido de un archivo mientras crece (ideal para logs). tail -f /var/log/syslog

Searching / Búsqueda

Herramientas para encontrar archivos, directorios o texto dentro de archivos de manera eficiente.

ComandoDescripción (EN/ES)Ejemplo de Uso
grep pattern file Search for pattern in files.Busca un 'patrón' de texto dentro de un archivo. grep "error" /var/log/nginx/error.log
grep -r pattern dir Search recursively for pattern in dir.Busca un patrón recursivamente en un directorio. grep -r "API_KEY" /etc/nginx/
command | grep Search for pattern in the output.Busca un patrón en la salida de otro comando. ps aux | grep "apache"
locate file Find all instances of file.Encuentra todas las instancias de un archivo (usa un índice). locate httpd.conf
find dir -name file Find files by name.Busca archivos por nombre en un directorio. find /home -name "*.jpg"

System Info / Información del Sistema

Comandos para obtener información sobre el hardware, el kernel y el estado general del sistema operativo.

ComandoDescripción (EN/ES)Ejemplo de Uso
date Show the current date and time.Muestra la fecha y hora actuales. date
cal Show this month's calendar.Muestra el calendario del mes actual. cal
uptime Show current uptime.Muestra cuánto tiempo ha estado encendido el sistema. uptime
whoami Who you are logged in as.Muestra el usuario con el que has iniciado sesión. whoami
uname -a Show kernel information.Muestra información detallada del kernel. uname -a
cat /proc/cpuinfo CPU information.Muestra información sobre la CPU. cat /proc/cpuinfo
df -h Show disk usage (human-readable).Muestra el uso del disco en formato legible. df -h
du -sh dir Show directory space usage.Muestra el espacio total usado por un directorio. du -sh /home/usuario
free -m Show memory and swap usage (in MB).Muestra el uso de memoria RAM y swap en Megabytes. free -m
man command Show the manual for command.Muestra el manual de ayuda para un comando. man ls

Process Management / Gestión de Procesos

Herramientas para monitorear y gestionar los procesos que se están ejecutando en el sistema.

ComandoDescripción (EN/ES)Ejemplo de Uso
ps aux Display all currently active processes.Muestra todos los procesos activos actualmente. ps aux | grep nginx
top Display all running processes (interactive).Muestra todos los procesos en ejecución de forma interactiva. top
kill pid Kill process id pid.Termina (mata) el proceso con el ID 'pid'. kill 12345
killall proc Kill all processes named proc.Termina todos los procesos con el nombre 'proc'. killall firefox
bg Lists stopped or background jobs.Lista y resume trabajos detenidos en segundo plano. bg
fg Brings the most recent job to foreground.Trae el trabajo más reciente al primer plano. fg

File Permissions / Permisos de Archivos

El comando chmod se utiliza para cambiar los permisos de lectura, escritura y ejecución de archivos y directorios. Se basa en un sistema octal.

ComandoDescripción (EN/ES)Ejemplo de Uso
chmod 755 file Change permissions to rwxr-xr-x.Cambia los permisos a lectura, escritura y ejecución para el dueño, y lectura y ejecución para el grupo y otros. chmod 755 mi_script.sh
chmod 644 file Change permissions to rw-r--r--.Cambia los permisos a lectura y escritura para el dueño, y solo lectura para el grupo y otros. Ideal para archivos web. chmod 644 index.html
chown user:group file Change ownership of file.Cambia el propietario y grupo de un archivo. chown www-data:www-data config.php

Network / Red

Comandos para diagnosticar problemas de red, transferir archivos y obtener información de dominios.

ComandoDescripción (EN/ES)Ejemplo de Uso
ping host Ping host and output results.Envía paquetes a un 'host' para comprobar la conectividad. ping google.com
whois domain Get whois information for domain.Obtiene información 'whois' de un dominio. whois google.com
dig domain Get DNS information for domain.Obtiene información DNS de un dominio. dig google.com
wget file Download file.Descarga un archivo desde una URL. wget https://example.com/file.zip
ssh user@host Connect to host as user.Conecta a un servidor remoto 'host' como el usuario 'user'. ssh admin@192.168.1.100
scp Secure copy (remote file copy).Copia archivos de forma segura entre hosts en una red. scp file.txt user@remote:/home/user/

Compression / Compresión

Comandos para comprimir y descomprimir archivos y directorios, optimizando el espacio de almacenamiento.

ComandoDescripción (EN/ES)Ejemplo de Uso
tar cf file.tar files Create a tar named file.tar.Crea un archivo 'tar' que agrupa varios archivos. tar cf archive.tar doc1.pdf doc2.pdf
tar xf file.tar Extract the files from file.tar.Extrae los archivos de un paquete 'tar'. tar xf archive.tar
tar czf file.tar.gz dir Create a tar with Gzip compression.Crea un archivo 'tar' comprimido con Gzip. tar czf project.tar.gz /var/www/project
tar xzf file.tar.gz Extract a tar using Gzip.Extrae un archivo 'tar' comprimido con Gzip. tar xzf project.tar.gz
gzip file Compresses file.Comprime un archivo y lo renombra a file.gz. gzip large_log_file.log

Shortcuts / Atajos del Teclado

Combinaciones de teclas que agilizan el trabajo en la línea de comandos.

AtajoDescripción (EN/ES)
Ctrl + C Halts the current command.Detiene/cancela el comando actual.
Ctrl + Z Stops the current command.Para el comando actual y lo envía al background.
Ctrl + D Log out of current session.Cierra la sesión actual (similar a 'exit').
Ctrl + R Type to bring up a recent command.Busca en el historial de comandos mientras escribes.
!! Repeats the last command.Repite el último comando ejecutado.
